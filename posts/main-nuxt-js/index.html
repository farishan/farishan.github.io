Main Nuxt.js

Ayo mainan Nuxt.js.

Nuxt.js itu framework-nya Vue.js. Vue.js itu framework-nya JavaScript.

Jadi, Nuxt.js itu framework-nya framework.

Bikin web pake JavaScript. Lebih gampang kalau pakai Vue.js (framework JavaScript). Lebih gampang lagi kalau pakai Nuxt.js (framework Vue.js).

js -> vue.js -> nuxt.js

2020/05/08

Website ini dibuat menggunakan Hugo.

Hugo mengklaim bahwa mereka adalah The worldâ€™s fastest framework for building websites, salah satu dari open source static site generator terpopuler. Terkenal dengan kecepatan tinggi dan fleksibilitas.

Mereka juga mengklaim menggunakan Hugo bisa membuat proses pengembangan website menjadi menyenangkan kembali, karena itulah saya memilih Hugo.

Selama ini saya belajar web development dengan HTML, CSS, dan JavaScript, lalu ke beberapa framework JS yang cukup terkenal dan punya nama seperti React dan Vue. Teknologi-teknologi tersebut saya pakai secara profesional dalam pekerjaan maupun sampingan untuk bersenang-senang.

Namun karena blogging adalah sesuatu yang masih asing untuk saya, mungkin ini saatnya mencoba teknologi baru yang memang lebih cocok untuk digunakan dalam pembuatan blog, seperti Hugo ini.

Jadi, selain bersenang-senang dengan framework baru, juga tetap belajar agar semakin luas pengetahuan tentang dunia website.
